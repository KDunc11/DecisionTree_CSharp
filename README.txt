Examples for file and the formats needed to use the tuple format builder are located in the UserInterface Examples folder,
to get there open the Decision Tree Folder above.

To locate the files that the tuple format builder saves traverse the directories as follows, from above open the Decision
Tree Folder, from there open the bin Folder, next open the Debug Folder, then finally the netcoreapp3.1 Folder which is where
the saved files are located.*

From within this folder
* Decision Tree -> bin -> Debug -> netcoreapp3.1
